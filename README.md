# Rekindled Loop

A tiny browser game made with Phaser 3.

You live a short life, make a few choices, die, and come back with a little of your old strength, intellect, and charm.

It is simple on purpose.

## What this is

- A text-based RPG with a reincarnation loop
- One Phaser scene
- Pixel-style placeholder art made in code
- Local save data with `localStorage`
- Static-site friendly for Netlify

## How to run

The easiest way:

1. Download the project
2. Open `index.html` in your browser

If your browser is picky about local files, use any small static server instead.

## How to deploy to Netlify

1. Zip the project folder or keep it as-is
2. Go to Netlify Drop
3. Drag the folder in

That is it. No build step. No backend. No setup.

## Files

- `index.html` - page shell and Phaser CDN
- `main.js` - game scene and reincarnation logic
- `events.js` - story events and choices
- `style.css` - simple page styling

## How to contribute

Homemade rules:

- Keep it simple
- Keep it playable fast
- No extra framework magic
- If you add something, try to keep the code easy to read

Good contributions:

- Fixing bugs
- Improving wording
- Adding small story branches
- Tweaking layout for mobile
- Cleaning up rough edges without overcomplicating things

If you make a change, a short note about what changed and why is plenty.
